# Архитектура

Система разделена на независимые слои:

- API: FastAPI routers в `backend/app/api`
- Services: бизнес-логика и расчеты в `backend/app/services`
- Repositories: работа с БД через SQLAlchemy ORM
- Models: SQLAlchemy 2.0 модели
- Schemas: Pydantic v2 контракты API и AI

ИИ-парсер работает через интерфейс `AIProvider`, поэтому Groq можно заменить на OpenAI или локальную модель без изменения бизнес-логики.

Отчеты не используют SQL, сгенерированный LLM. Вопрос пользователя классифицируется безопасным сервисом, затем выполняются заранее описанные ORM-запросы.
